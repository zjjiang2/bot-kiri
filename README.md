### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/zjjiang2/bot-kiri.git
   cd bot-kiri
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Create a `.env` file and add your bot token:

   ```env
   DISCORD_TOKEN="your_bot_token"
   CLIENT_ID = "your_client_id"
   ```

4. Run the bot:

   ```bash
   node ./app.js
   ```
